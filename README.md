# dotfiles

Collection of various configuration files.

## Linux Setup
Running Fedora + GNOME.

**Extensions:**

* Blur my Shell
* Caffeine
* Clipboard History
* Dash to Panel
* GSConnect
* Just Perfection
* Vitals

**Eyecandy:**

* Adwaita + adw-gtk3 for legacy apps
* Papirus icons
* Adwaita cursors
* Fira Sans for UI
* IBM Plex Mono for monospace
