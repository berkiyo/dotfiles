# dotfiles

Collection of various configuration files.

## Linux Setup
Running Fedora + GNOME.

**Extensions:**

* Blur my Shell
* Caffeine
* Clipboard History
* Dash to Panel
* GSConnect
* Vitals

**Eyecandy:**

* Adwaita + adw-gtk3 for legacy apps
* Papirus icons
* Adwaita cursors
* Cantarell for UI
* 0xProto for monospace
